# docker-to-spring-example

   |옵션|설명|
    |:---:|:---:|
    |-d|detached mode. 백그라운드모드를 의미|
    |-p|호스트와 컨테이너의 포트를 연결(포워딩)|
    |-v|호스트와 컨테이너의 디렉토리를 연결(마운트)(볼륨)|
    |-e|컨테이너 내에서 사용할 환경변수를 설정함.|
    |-name|컨테이너의 이름을 설정|
    |-rm|컨테이너의 프로세스가 종료될 시 자동제거|
    |-it|터미널 키보드 system.in 옵션|
    |-link|컨테이너를 연결한다 ([컨테이너명:별칭])|
